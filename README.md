# This is repo for my vim config

## setup 

### clone Vundle
```
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

### get .vimrc
```
wget https://raw.githubusercontent.com/otzhora/vim-config/master/vimrc -O ~/.vimrc
```

### install plugins 

```
vim
:PluginInstall
```
